### 2. Типографика hero (только домашний variant)

В `HeroCard.tsx` (ветка без `title`):

- **Имя «Андрей Майнгардт» → eyebrow над H1:**
  - `text-transform: uppercase`
  - `letter-spacing: 0.1em`
  - `font-weight: 500`
  - размер `clamp(13px, 1.05vw, 15px)` (eyebrow крупнее текущего mute-варианта в визуальной системе сайта, но всё ещё подчинённый H1)
  - цвет — `#9a9aaa` (text-secondary) — приглушённый, читаемый
  - расположить НАД аватаром? Нет — оставить порядок: аватар → eyebrow-имя → H1 → описание → регалии → соцсети. Сдвиг текущий `mt-7` уменьшить до `mt-6`.

- **H1 «Эксперт по влиянию ИИ…» — в 2 строки:**
  - уменьшить: `font-size: clamp(32px, 4.4vw, 48px)` (было `clamp(36px, 5.5vw, 56px)`)
  - `line-height: 1.1`
  - `letter-spacing: -0.02em`
  - `max-width: 22ch` чтобы естественно ложилось в 2 строки на desktop
  - добавить `text-wrap: balance` (через inline `style={{ textWrap: 'balance' }}`)
  - цвет `#f0f0f5` сохранить

- **Сократить контраст имя↔H1:**
  - Имя — eyebrow стиль (uppercase, разрядка), визуально читается как метка одной системы с H1, а не как «второй заголовок».
  - H1 уменьшен с 56→48px max; разрыв размеров стал гармоничнее.

- **Вертикальный ритм** (равномерный, чуть больше воздуха перед регалиями):
  - аватар → eyebrow-имя: `mt-6`
  - eyebrow → H1: `mt-3`
  - H1 → описание: `mt-5`
  - описание → регалии: `mt-7` (больше воздуха здесь)
  - регалии → соцсети: `mt-8`
